# Archive — do not build from here

Everything under **`archive/`** is **read-only history**: old PRD bundles and completed task lists. It is intentionally **not** part of active product or engineering workflows.

### Glu AI — what to use instead

| Use this | Purpose |
|----------|---------|
| [`apps/GluAI/design.md`](../apps/GluAI/design.md) | Product + design tokens + flows |
| [`apps/GluAI/screens_updated.md`](../apps/GluAI/screens_updated.md) | Screen / interaction brief (subordinate to `design.md`) |

### Layout

| Path | Contents |
|------|----------|
| [`prd-glu-ai/`](prd-glu-ai/) | Legacy oneshot PRD Markdown used to scaffold the first build |
| [`tasks/`](tasks/) | Historical Glu redesign / ship checklists |

If something in **`archive/`** disagrees with **`apps/GluAI/design.md`**, **`design.md` wins**.
